Style Guide

The style guide is a living set of HTML documents that defines the design and coding rules to determine how the website is structured today, as well as to ensure consistency for future updates.

If you have any questions, submit an issue in our GitHub repo.

Elements

These are the smallest building blocks for an interface, such as headings, buttons, form input fields and icons.

More abstract elements like the font stack and color palette are also included here.

View

Patterns

These are groups of elements, that when combined, function together as a unit.

Examples of these common snippets of markup are navigation, tabs or a list of articles.

View

Forms

These are combinations of normal content and controls such as labels, inputs, and checkboxes.

For example, a search form consisting of a label, text input and submit button.

View

Grid

This is a combination of containers and columns which provides structure for content.

Column sizes are specified in fractions and take up the full width available on narrow screens.
