Style Guide

Elements

Elements are the basic building blocks of the interface. They consist of headings, buttons, icons and more abstract things like color palettes and font stacks.

Explore

Patterns

Patterns are groups of elements that are combined to function together as a unit. For example, a header containing a logo and navigation for the website.

Explore

Forms

Forms contain interactive controls that are used to submit information to a web server. These include inputs, checkboxes and submit buttons, etc.

Explore

Templates

Templates are made up from elements, patterns and forms to create layouts within the website. They provide structure and context for the content.
